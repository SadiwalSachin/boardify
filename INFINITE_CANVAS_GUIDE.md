## ğŸ–±ï¸ How to Use Panning

### Method 1: Spacebar + Drag (Recommended)
1. **Hold down the Spacebar** key
2. You'll see a blue indicator: "Pan Mode Active - Click and drag to move canvas"
3. **Click and drag** anywhere on the canvas to pan
4. **Release Spacebar** to return to normal drawing mode

**Visual Feedback:**
- Cursor changes to "grab" icon when spacebar is held
- Blue indicator appears at bottom-left showing pan mode is active

### Method 2: Middle Mouse Button
1. **Click and hold the middle mouse button** (scroll wheel button)
2. **Drag** to pan the canvas
3. **Release** to stop panning

### Method 3: Two-Finger Drag (Trackpad)
- On trackpad, use **two-finger drag** to pan
- Works on both Mac and Windows trackpads

## ğŸ› Troubleshooting

### Pan Mode Won't Activate
- **Check:** Make sure you're not editing text (press Escape first)
- **Try:** Release and press Spacebar again
- **Alternative:** Use middle mouse button instead

### Canvas Feels "Stuck"
- **Solution:** Click the Reset Zoom button (maximize icon)
- **Check:** Make sure you're not in text editing mode

### Zoom Too Fast/Slow
- **Adjust:** Use the +/- buttons for controlled zooming
- **Alternative:** Use trackpad pinch for smoother control

### Lost Your Drawing
- **Don't panic:** Your drawing is still there, just pan around to find it
- **Quick fix:** Click Reset Zoom button to return to center
- **Prevention:** Use zoom out to see the whole canvas before panning
